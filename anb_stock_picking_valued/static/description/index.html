Stock Picking Valued

Stock picking

New fields on stock picking model:

  • Pricelist.
  • Currency.
  • Payment Term.
  • Customer/Supplier Reference.
  • Fiscal Position.
  • Untaxed Amount.
  • Taxes.
  • Total.

Stock move

New fields on stock move model:

  • Picking type code (hidden).
  • Unit Price.
  • Subtotal.
  • Taxes.
  • Total.
  • Price Reduce.
  • Price Reduce Tax inc.
  • Price Reduce Tax excl.
  • Discount (%).
  • Currency.

Connection between models

Fields added to picking model can work alone and can got their data from sale/purchase orders.

Delivery slip report modified

The delivery slip report does not show the operations anymore, now it always shows the moves which makes more sense.

Installation

To install this module, you need to:

  • You must have installed or available to install the module purchase.
  • You must have installed or available to install the module stock.

Usage

To use this module, you need to:

  • Go to Sales > Sales > Quotations / Sale Orders, and create a new order.
  • Confirm the sale order and go to see the generated picking.
  • Check that the data is rightly moved from the sale order model to the picking one.
  • Go to Purchases > Purchase > Requests for Quotations / Purchase Orders, and create a new order.
  • Confirm the purchase order and go to see the generated picking.
  • Check that the data is rightly moved from the purchase order model to the picking one.
  • Go to Warehouse > Operations > All Operations, and create a picking.
  • Check that the data is rightly updated just like if you were creating a sale/purchase order, depending on the picking type.
